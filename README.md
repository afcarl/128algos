# 128 algorithmes

## TryAlgo in Paris

Pour lancer le notebook Jupyter, entrez les commandes suivantes :

    python3 -m venv venv
    . venv/bin/activate
    pip install -r requirements.txt
    jupyter notebook

Une fenêtre vers http://localhost:8888 devrait se lancer, et vous pourrez y exécuter les commandes.
